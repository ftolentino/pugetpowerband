### Image Specifications

- **File name:** `og-image.jpg` (or `og-image.png`)
- **Location:** Place in this folder (`/public/`)
- **Dimensions:** 1200 x 630 pixels
- **Format:** JPG or PNG
- **File size:** Keep under 1MB for fast loading

### Design Tools

**Free Options:**
- **Canva** - canva.com (has templates for social media)
- **Figma** - figma.com (professional design tool)
- **Photopea** - photopea.com (free Photoshop alternative)

**Quick Template:**
1. Go to Canva.com
2. Search "Facebook Post" or create custom 1200x630px canvas
3. Add your band photo as background
4. Add text overlay with band name
5. Download as JPG

### After Creating

1. Save as `og-image.jpg`
2. Place in `/public/` folder (this directory)
3. The image will automatically appear when sharing your site!
